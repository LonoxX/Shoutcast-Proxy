# Shoutcast-Proxy

Proxy für Shoutcast server

# Installation

1. Erstelle eine ordner stream
2. Lade alles auf den Webspace hoch
3. Platzhalter "IP / PORT" entsprechend ändern [index.php]
4. Fertig Öffne einen Browser und rufen die URL auf

```php
  $server = "IP"; // Zeile 12
  $port = "PORT"; // Zeile 13
```
